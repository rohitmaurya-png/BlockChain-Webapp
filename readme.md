Installation of truffle on windows machine 

1.```npm install -g truffle```
2.```npm install -g npm```
3.```npm install -g ethereumjs-testrpcnpm```
4.```install -g -production windows-build-tools```

To start the project on localhost
1.```truffle compile```
2.```truffle migrate```
3.```npm install```
4.```npm start```
